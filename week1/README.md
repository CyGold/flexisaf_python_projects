# Student Score Calculator using python

A Python program that demonstrates advanced function usage including variable-length arguments,
keyword arguments, argument unpacking, and variable scope. 

---

##  Features

- Accepts a flexible number of student scores using `*args`
- Supports optional configuration such as bonus points and weighting via `**kwargs`
- Demonstrates argument unpacking from lists, tuples, and dictionaries
- Uses global and local variables to illustrate scope
- Processes multiple students in a single run
- Outputs clear, readable student performance reports

---

##  Concepts Demonstrated

This project showcases core Python concepts such as:

- Variable-length arguments (`*args`)
- Keyword arguments (`**kwargs`)
- Argument unpacking (`*` and `**`)
- Global vs local variable scope
- Function composition and reusability
- Clean, readable program structure
